<TipBanner>
  <b>핵심 — </b>아이솔레이션은 <b>전압 격리 (kV 급 견디기)</b> 와 <b>전류 차단 (DC 0 A)</b> 을 동시에. 단순한 cap / R 분리는 격리가 아닙니다 (DC 흐름 막아도 AC 누전 가능). 진짜 격리는 <b>물리적 갭 + 인증된 부품</b>.

<div class="mt-6">
  <Card id="why" title="1. 왜 아이솔레이션인가? — 4가지 동기">
    <div class="prose prose-slate max-w-none">
      <p>거의 모든 아이솔레이션 적용은 다음 4가지 중 하나 또는 조합:</p>
      <ol>
        <li>
          <b>안전 (Safety)</b> — AC mains, 의료기기 환자 누전 (patient leakage), 고압 산업 장비. 사용자 / 기술자가 만질 수 있는 부분 (저전압 측) 과 위험 전압 (mains, kV) 을 절연 → 감전 / 화재 방지. <b>법적 요건</b> (UL, IEC 60601, KC 등).
        </li>
        <li>
          <b>전압 차 (Different ground potential)</b> — 두 회로의 GND 가 수십 V 이상 차이 나는 경우. 예) 100 V 모터 드라이브의 high-side gate driver, 16-cell 배터리 stack 의 high cell vs MCU GND. 두 도메인을 직접 연결하면 거대한 short / 손상.
        </li>
        <li>
          <b>그라운드 루프 / 노이즈 차단</b> — 두 시스템이 별도 ground 인데 통신선으로 연결되면, 미세한 ground 전위차로 잡음 / 부식. 산업 RS-485 / CAN, USB 측정 장비 등 → isolator 로 GND 단절.
        </li>
        <li>
          <b>회로 보호</b> — 한쪽 회로가 손상되어도 다른 쪽으로 전파 안 됨. 비싼 host 와 험한 환경의 sensor 사이.
        </li>
      </ol>
    </div>

<div class="mt-6">
  <Card id="methods" title="3. 어떤 방식으로 — 4가지 isolator 종류">
    <IsolationBarrier />
    <div class="prose prose-slate max-w-none mt-3">
      <ol>
        <li>
          <b>① Optocoupler</b> (LED + photo-transistor) — 전통적, 싸고 단순. 단점: <b>속도 느림</b> (수 100 kbps 이하), CTR (전류 전송비) 산포 ±50%, 시간에 따라 열화. AC mains 피드백, 단순 디지털 신호 격리에 여전히 흔함.
        </li>
        <li>
          <b>② 디지털 isolator</b> (capacitive: TI ISO78xx / Silicon Labs Si86xx, magnetic: Analog Devices ADuM) — 칩 안에 작은 cap / 트랜스로 신호 전달. <b>매우 빠름</b> (Mbps~Gbps), CTR 안정, 수명 길고, 채널 다수. <b>현대 isolation 의 표준</b>. 비용 옵토의 2~3배.
        </li>
        <li>
          <b>③ 트랜스포머</b> — 신호 + 전력 동시 전달. 이더넷의 마그네틱, USB isolator 안의 코일, isolated DC-DC 의 출력 단까지 다 트랜스 기반. <b>전력 전달도 가능</b> 하다는 게 다른 방식과의 큰 차이.
        </li>
        <li>
          <b>④ Isolated DC-DC</b> — 격리된 보조 전원. isolator 의 secondary 측을 동작시키려면 보통 별도 전원 필요. 작은 power module (Murata MEU1, RECOM RxxP21) 또는 직접 transformer + flyback 설계.
        </li>
      </ol>
      <p>
        <b>실무 패턴</b> — 신호 isolator (디지털 isolator) + 격리 전원 (DC-DC isolator) 두 부품을 같이 쓰는 게 일반적. ADuM5410 / ISO7841 처럼 <b>두 기능이 한 칩</b> 인 부품도 많음.
      </p>
    </div>
  </Card>
</div>

<div class="mt-6">
  <Card id="ratings" title="4. 격리 등급 — basic / reinforced / 인증 표준">
    <div class="prose prose-slate max-w-none">
      <p>
        부품 / 제품 격리에는 등급이 있고, 등급에 따라 요구되는 <b>creepage / clearance</b> 가 다릅니다:
      </p>
      <ul>
        <li><b>Functional</b> — 단순 신호 분리. 안전 의미 없음. 그라운드 루프 차단용.</li>
        <li><b>Basic</b> — 1중 안전. 한 단계의 절연이 정상 동작 시 안전 보장. 산업 일반 + 비의료.</li>
        <li><b>Reinforced</b> (= 2× basic) — 두 겹의 절연 또는 강화된 한 겹. 사용자가 손 댈 수 있는 mains 인터페이스, 의료기기 (환자 응용 부분).</li>
        <li><b>Double</b> — basic + basic 두 개를 직렬로. reinforced 와 같은 안전 등급이지만 부품 두 개 사용.</li>
      </ul>
      <p><b>주요 인증 표준</b>:</p>
      <ul>
        <li><b>UL 1577</b> (US 일반), <b>IEC 60747-17</b> (디지털 isolator 부품)</li>
        <li><b>IEC 60601</b> (의료기기) — 환자 응용 부분 매우 엄격, 4 kV 이상</li>
        <li><b>IEC 60950 / 62368-1</b> (IT/AV 장비)</li>
        <li><b>UL 508A</b> (산업 제어반)</li>
      </ul>
      <p>
        부품 데이터시트의 <b>V_IORM</b> (continuous working voltage) 과 <b>V_IOTM</b> (transient peak) 두 값을 봐야. V_IORM 이 시스템의 working voltage 보다 충분히 커야 함.
      </p>
    </div>
  </Card>
</div>

<div class="mt-6">
  <Card id="layout" title="5. 레이아웃 — slot, plane 분리, 부품 straddling">
    <IsolationLayout />
    <div class="prose prose-slate max-w-none mt-3">
      <p><b>철칙</b>:</p>
      <ol>
        <li>
          <b>Plane / GND 도 분리</b> — 두 도메인의 GND plane 을 PCB slot 으로 물리적 분리. 같은 plane 위에 두면 isolator 의미 사라짐.
        </li>
        <li>
          <b>Isolator 부품만 slot 을 가로지름</b> — 다른 신호 / 전원 트레이스가 slot 을 횡단하면 안됨. <a href="/return-paths/basics#slit" class="text-indigo-600 underline decoration-indigo-200 hover:decoration-indigo-600">return path</a> 도 깨지고 격리도 깨짐.
        </li>
        <li>
          <b>Creepage 거리 확보</b> — 위 계산기로 산출한 mm 값보다 conductor 간 표면 거리 충분히 확보. 솔더마스크 위 실크스크린도 거리 줄임 → 깨끗한 영역.
        </li>
        <li>
          <b>Isolator 의 V_IORM 확인</b> — 부품 자체 격리 등급 (예: ISO7841 = 5 kV reinforced) 이 시스템 요구 등급보다 같거나 커야.
        </li>
        <li>
          <b>Y-cap 으로 mains 격리</b> — AC 입력 isolation 회로는 1차/2차 사이에 안전 인증 (Y1/Y2) cap 으로 노이즈 통과. 일반 cap 사용 금지.
        </li>
        <li>
          <b>Standoff / 마운팅 점검</b> — 두 도메인이 metal chassis / 마운팅 나사 / 케이스 통해 우회 연결되지 않게.
        </li>
      </ol>
    </div>
  </Card>
</div>

<div class="mt-6">
  <Card id="pitfalls" title="6. 흔한 실수">
    <div class="prose prose-slate max-w-none">
      <ol>
        <li><b>Plane 미분리</b> — isolator 부품은 있는데 plane 이 두 도메인에 공통. 사실상 격리 안 됨.</li>
        <li><b>Slot 위 다른 신호</b> — return path 가 plane 우회 → 큰 loop area + EMI + 격리 우회.</li>
        <li><b>Creepage 부족</b> — 250V mains 인데 conductor 간 1mm 만 띄움. 인증 시험에서 fail / 습도 높을 때 spark.</li>
        <li><b>Y-cap 없이 일반 cap</b> — AC noise 가 타고 들어옴. 안전 인증 cap (X1/X2/Y1/Y2 표시) 만 사용.</li>
        <li><b>Isolator 의 secondary 전원 빠짐</b> — 디지털 isolator 는 양쪽 다 전원 필요. secondary 측에 별도 isolated DC-DC 또는 통합 칩 사용.</li>
        <li><b>나사 / 마운팅 통한 우회 GND</b> — 두 영역의 PCB 마운팅 홀이 같은 metal frame 에 닿으면 격리 우회.</li>
        <li><b>Reinforced 라고 표기된 부품인데 단일 chip basic</b> — V_IORM 만 보지 말고 isolation type (basic / reinforced) 도 확인.</li>
        <li><b>고전압 surge 무시</b> — 격리 부품은 working voltage 외에 surge / transient (V_IOTM) 도 견뎌야. 산업 환경은 4 kV 이상 surge 일반.</li>
      </ol>
    </div>
  </Card>
</div>
